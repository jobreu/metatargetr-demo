# metatargetr-demo

Jupyter Lab [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/jobreu/metatargetr-demo/main)

RStudio [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/jobreu/metatargetr-demo/main?urlpath=rstudio)

Demo repo for collecting ad data from *Meta* and *Google* with the `R` package [metatargetr]((https://favstats.github.io/metatargetr/)

Created for the masters-level seminar "Erforschung von politischer Online-Kommunikation mit digitalen Methoden" ("Studying political online communication with digital methods") at the University of Duisburg-Essen (winter term 2025/2026).
